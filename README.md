# 📊 SaaS Sales Analysis Project

## 📌 Overview

This project analyzes SaaS sales data to evaluate business performance, customer behavior, and revenue trends using SQL.

The goal is to extract actionable insights that help understand how the business is performing and where improvements can be made.

---

## ❓ Problem Statement

Businesses often struggle to understand:

* How revenue is growing over time
* Which products drive the most sales
* Customer purchasing behavior
* Operational performance (orders, inventory, etc.)

This project answers these questions using structured SQL analysis.

## 📊 Key Metrics & Analysis

The project includes multiple layers of analysis:

### 🔹 Business Overview

* Total Revenue
* Total Orders
* Customer Activity
* Average Order Value

### 🔹 Orders Analysis

* Order trends over time
* Purchase patterns

### 🔹 Product Analysis

* Top-selling products
* Product performance

### 🔹 User Analysis

* Active users
* User behavior

### 🔹 Inventory Analysis

* Stock levels
* Inventory distribution

---

## 💡 Key Insights

* Revenue is driven by a subset of active users
* Certain products contribute disproportionately to total sales
* Order trends indicate potential growth over time
* Inventory levels highlight possible stock risks
